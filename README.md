# Vinculi

Vinculi project contains all stuff for vinculi website and api:  
  - ars_magica: Access to Ars Magica database  
  - vinculi_db: Access to Vinculi postgres database  
  - vinculi_api: The vinculi api and access to Vinculi Neo4J database  
  - vinuli_web: The website
